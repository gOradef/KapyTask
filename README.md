### Определение
KapyTask - это приложение, которое позволяет управлять заданиями среди различных дисциплин.

## Проблема
Todoist, конечно, удобен, однако у него есть такие недостатки, как нельзя привязать задание к конкретной дисциплине.
Также пользователь сталкивается с тем, что при создании новой задачи нужно установить: дисциплину, дедлайн, дату выполнения, где **дедлайн можно установить по умолчанию на ближайшее занятие в расписании студента.**

### Функции
1. Хранение и изменение расписания пользователя
2. Хранение и изменение списка дисциплин пользователя
3. Создание задач, которые можно:
    1. Прикрепить к дисциплине
    2. Установить дедлайн, подтянув ближайшее занятие по данной дисциплине
    3. Установить планированию дату для выполнения
    4. Установить приоритеты выполнения (красный - серый)
### Перспективы
- Экспорт/Импорт конфигурации пользователя, чтобы внедрить среди одногруппников (атвичаю, майнера не будет 1000% =) )
- Система виджетов на главном экране для быстрого управления задачами
- EventMessager, сервис уведомлений для
    - напоминаний в день планируемого выполнения задачи
    - кастомных уведомлений по предусмотрению пользователя
- Синхронизация с `calendar.google.com`
    - Синхронизация задач
    - импорт / экспорт задач, расписания среди пользователей
- Создание совместных задач (проектов) между несколькими пользователями

### Требования к MWP
- Локальное приложение
- Наличие вышеописанных функций
